# Calculator

A simple calculator web app built with HTML, CSS, and JavaScript.

## Files

- `index.html` — app structure and UI
- `style.css` — styling for calculator layout
- `script.js` — calculator logic and interaction

## Usage

1. Open `index.html` in a browser.
2. Use the calculator buttons to perform basic arithmetic operations.
